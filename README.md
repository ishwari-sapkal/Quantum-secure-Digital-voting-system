# 🗳️ Quantum Secure Digital Voting System

A secure web-based voting platform developed using Flask and MongoDB that enables users to cast votes electronically while ensuring data confidentiality, integrity, and transparency. The system incorporates encryption techniques to protect voting data and prevent unauthorized modifications.

## 🚀 Features

* Secure voter registration and authentication
* Encrypted vote storage
* Prevention of duplicate voting
* Real-time vote counting and result management
* User-friendly web interface
* Admin dashboard for election monitoring
* MongoDB database integration

## 🛠️ Tech Stack

* Python
* Flask
* MongoDB
* HTML
* CSS
* JavaScript

## 🎯 Project Objective

The objective of this project is to provide a secure and efficient digital voting solution that minimizes manual effort while ensuring transparency and data security throughout the voting process.

## 📂 Installation

```bash
git clone <repository-url>
cd Quantum-secure-Digital-voting-system
pip install -r requirements.txt
python app.py
```
