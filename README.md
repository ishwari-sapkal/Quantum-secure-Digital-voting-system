# Digital Voting System

## Overview

The Digital Voting System is a secure web-based application developed using Flask and MongoDB. It enables users to cast votes electronically while maintaining vote confidentiality and data integrity through encryption techniques.

## Features

* Secure voter registration and authentication
* Encrypted vote storage
* Prevention of duplicate voting
* Real-time vote management
* User-friendly interface
* Secure database integration with MongoDB

## Technology Stack

* Python
* Flask
* MongoDB
* HTML
* CSS

## Project Objective

The primary goal of this project is to provide a secure, transparent, and efficient digital voting platform that minimizes manual effort while protecting voter information.

## Installation

1. Clone the repository.
2. Install required dependencies:

   ```bash
   pip install -r requirements.txt
   ```
3. Configure MongoDB connection settings.
4. Run the application:

   ```bash
   python app.py
   ```
